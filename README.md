# Automation_Python

## About This Project

This is a bootcamp project created to practice and learn more about **automation of tasks and bots with Python**.

The goal is to build small, focused scripts and bots that automate repetitive tasks, helping to develop real-world Python skills in the area of automation.

## What You'll Find Here

- Scripts for automating everyday tasks
- Bots built with Python
- Hands-on exercises from the bootcamp curriculum

## Technologies

- **Python** — main programming language used throughout the project

## Purpose

This repository serves as a learning playground for exploring Python automation libraries and techniques. Each project is an opportunity to experiment, make mistakes, and grow as a developer.
